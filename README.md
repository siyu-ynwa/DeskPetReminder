## Desk Pet Reminder 🐶
A cute desktop health tool that reminds you to take breaks from sitting, featuring an animated puppy companion.

## Features
🐕 Animated Puppy – Changes expression as sitting time increases

⏰ Custom Timer – Set your own reminder time (default 30 min)

🖱️ Drag & Drop – Move the puppy up/down with mouse

🎨 Resize – Adjust puppy size via right-click menu (10%-300%)

⏸️ Pause/Resume – Pause timer anytime

🖼️ Custom Background – Use your own image for popup

📦 Portable – Single .exe file, no installation needed

## Quick Start
1.Download & Run

Download 久坐提醒.exe from Releases

Double-click to run

Enter sitting time in minutes and start

2.Run from Source

git clone https://github.com/siyu-ynwa/DeskPetReminder.git

cd DeskPetReminder

pip install pillow tkinter 

python main.py

## Controls

Action: Effect

Left Drag: Move puppy vertically

Right Click: Open menu (Pause/Resume, Resize, Exit)

"OK" in Popup: Reset timer

"Exit" in Popup / X Button: Close program

## Build to EXE

Double click build.bat

## Requirements
Windows 10/11
